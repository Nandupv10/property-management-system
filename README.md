# 🏠 Property Management System

A full-stack Property Management System built using React, Node.js, Express.js, and MongoDB. The application allows tenants to submit maintenance requests while property owners can manage and monitor requests through a dashboard.

## Features

- Tenant Dashboard
- Owner Dashboard
- Maintenance Request System
- Login Page
- Amenities Booking
- Booking Management
- REST API using Express.js
- MongoDB Database
- Responsive UI

## Tech Stack
